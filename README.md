# Fry

Juego que pone a prueba tus reflejos y te hace salirte.

## Cómo usarlo/Descargarlo

1. Descarga o clona el proyecto
2. Abre el archivo `index.html`
3. Disfruta y estresate

## Requisitos

- Navegador web (Chrome, Firefox, etc.)

## Estructura del proyecto

- front{
    - index.html → página principal (solo para ejecutar)
    - style.css → estilos (no meterte con ellos)
    - script.js → lógica (no meterte con ellos)
    }
- media (no tocar)
- imagenes (no usmear)
- server (no tocar)
